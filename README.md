rocket-launcher
===============

Application to launch a rocket from a smart watch using Arduino

Hardware
========

* Arduino UNO
* Bluetooth shield (http://www.seeedstudio.com/wiki/Bluetooth_Shield)
* Relay shield (http://www.seeedstudio.com/wiki/Relay_Shield_V2.0)
* Smartphone
* Smartwatch
